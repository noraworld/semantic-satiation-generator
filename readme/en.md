# Semantic Satiation Generator

[日本語 README](./ja.md)

Semantic Satiation Generator is a simple static site for quickly experiencing semantic satiation by filling the screen with the same word or phrase over and over.

## Features

- Fill the entire screen with a repeated word or short phrase
- Adjust font size, weight, letter spacing, and text spacing
- Change text color, background color, and font family
- Live preview before generating the full-screen view
- English and Japanese UI with browser-language detection and manual switching
- Static HTML, CSS, and JavaScript only, suitable for GitHub Pages

## Usage

1. Open `index.html` in a browser, or deploy the repository with GitHub Pages.
2. Enter a word or phrase.
3. Adjust the visual options if needed.
4. Click `Fill the Screen` or press Enter.
5. Use the `×` button or `Esc` to return to the input screen.

## Deployment

This project is a plain static site. To publish it with GitHub Pages, serve the repository root from the `main` branch.
